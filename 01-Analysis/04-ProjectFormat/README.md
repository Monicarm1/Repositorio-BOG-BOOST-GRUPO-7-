# Formato proyecto 

En el siguiente documento se desarrolla el análisis y diseño de un sistema de información para mejorar la gestión y comercialización de los negocios del Mercado de las Pulgas de San Alejo en Bogotá. Se presentan el problema identificado, los objetivos del proyecto, el marco teórico que lo respalda y la metodología utilizada para la recolección y análisis de la información. Finalmente, se describe la propuesta tecnológica planteada como solución a las necesidades detectadas.
